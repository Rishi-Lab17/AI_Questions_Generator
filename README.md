## 📌 Overview

An **AI-powered web application** that generates intelligent question papers from uploaded PDFs, textbooks, notes, and question banks. Perfect for educators, students, and self-learners.

## ✨ Features

| Feature | Description |
|---------|-------------|
| 📚 **3 Separate Upload Sections** | Textbooks, Notes, Question Banks |
| 🤖 **AI Topic Extraction** | Automatically identifies key topics |
| 📝 **Smart Answer Generation** | Answers extracted from your uploaded notes |
| 🎯 **Module Distribution** | Equal or Weightage-based distribution |
| 📊 **Difficulty Levels** | Easy/Medium/Hard with custom percentages |
| 💰 **Mark Distribution** | 2, 5, 10 mark questions |
| 📈 **Difficulty Heatmap** | Visual representation of question distribution |
| 💾 **Download as TXT** | Save question papers locally |
| 📱 **Mobile Responsive** | Works on any device |
| 🕐 **Time-based Greeting** | Personalized welcome message |

## 🛠️ Tech Stack

- **Backend:** Flask (Python)
- **Frontend:** HTML, CSS, JavaScript
- **AI Features:** Custom NLP algorithms for topic extraction and answer generation

## 🚀 Installation & Usage

### Prerequisites
- Python 3.9 or higher
- pip package manager

### Steps to Run

```bash
# 1. Clone the repository
git clone https://github.com/YOUR_USERNAME/AI-PDF-Question-Generator.git
cd AI-PDF-Question-Generator

# 2. Install dependencies
pip install flask

# 3. Run the application
python app.py

# 4. Open browser and go to
http://127.0.0.1:5000
